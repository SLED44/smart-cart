SmartCart Meal Planner — Recipe JSON schema
============================================

Schema with type annotations (use as a reference / Claude.ai snippet):

{
  "title":             "Required, string",
  "source_url":        "Optional, the original recipe URL or empty string",
  "image_url":         "Optional, image URL (CDN) or empty string",
  "servings_original": 4,
  "ready_in_minutes":  30,
  "prep_minutes":      10,
  "cook_minutes":      20,
  "cuisines":          ["american"],
  "diet_tags":         [],
  "dish_types":        ["main course"],
  "equipment":         ["air_fryer", "sheet_pan"],
  "proteins":          ["beef"],
  "carbs":             ["bread"],
  "ingredients": [
    {
      "name":          "ground beef (80/20)",
      "amount":        1.5,
      "unit":          "lbs",
      "aisle":         "Meat",
      "original_text": "1.5 lbs 80/20 ground beef"
    }
  ],
  "instructions": [
    { "step_number": 1, "text": "Preheat air fryer to 425°F for 5 min." }
  ]
}


Allowed enum values
-------------------
proteins:  beef, pork, chicken, fish, lamb, shrimp, plant
carbs:     rice, pasta, bread, grain, potato, salad
cuisines:  american, italian, mexican, japanese, korean, vietnamese,
           thai, chinese, mediterranean, greek, middle_eastern,
           moroccan, indian   (add new ones as needed)
equipment: lowercase_with_underscores —
           air_fryer, sheet_pan, oven, stovetop, skillet, dutch_oven,
           instant_pot, slow_cooker, grill, blender, food_processor

Notes
-----
- Use the in-app **📝 Paste a recipe** screen (Meal Planner tab → Settings)
  to validate + save. The Claude.ai prompt template (also in that screen)
  embeds this schema plus household preferences (mild spice, air-fryer-
  first, household size 4, buy-don't-make-sauces).

- ingredient `unit` accepts: "lb", "lbs", "oz", "g", "cup", "tbsp", "tsp",
  "ml", "count", "slices", "" (empty for things like "1 onion"). Match
  exactly across recipes so the grocery aggregator can sum quantities.

- ingredient `aisle` uses Spoonacular-style labels: Meat, Seafood, Produce,
  Dairy, Pantry, Frozen, Bakery, Beverages, Spices, "Canned and Jarred",
  Baking, "Cleaning Products". The grocery aggregator maps these to
  SmartCart's 8-category schema.

- `instructions[].step_number` is 1-indexed and must be sequential.

- All fields outside `title`, `servings_original`, `ready_in_minutes`,
  `ingredients`, and `instructions` are optional but recommended — the
  rules engine + planner won't crash if they're missing, just won't
  score those dimensions.
